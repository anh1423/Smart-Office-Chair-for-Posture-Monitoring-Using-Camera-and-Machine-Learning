üßò Posture Detection

POST /api/predict üîí API Key Required

Receives data from 7 FSR sensors and returns posture recognition result.

Request Body (JSON):
sensor1 float required
Sensor 1 value
sensor2 float required
Sensor 2 value
sensor3...sensor7 float required
Remaining sensor values
Example Request:
curl -X POST http://localhost:5000/api/predict \
  -H "Content-Type: application/json" \
  -H "X-API-Key: your-api-key-here" \
  -d '{
    "sensor1": 100,
    "sensor2": 150,
    "sensor3": 120,
    "sensor4": 180,
    "sensor5": 95,
    "sensor6": 110,
    "sensor7": 140
  }'
Response (200 OK):
{
  "label": "Correct_posture",
  "confidence": 0.95,
  "warning": false,
  "mode": "auto",
  "timestamp": "2025-12-17T12:30:00",
  "metadata": {
    "camera_used": false,
    "sensor_confidence": 0.95
  }
}
GET /api/latest üîì Public

Response (200 OK):

üîã Battery Monitoring

POST /api/battery üîí API Key Required

Receive battery data from ESP32 and save to database.

Request Body (JSON):
voltage float required
Battery voltage (V)
percentage float required
Battery percentage (0-100)
level string required
Battery level: critical, low, medium, high
Example Request:
curl -X POST http://localhost:5000/api/battery \
  -H "Content-Type: application/json" \
  -H "X-API-Key: your-api-key-here" \
  -d '{
    "voltage": 3.85,
    "percentage": 70.8,
    "level": "medium"
  }'
GET /api/battery/latest üîí Login Required

Get latest battery information.

GET /api/battery/history?hours=24 üîí Login Required

Get battery history for the last N hours.

üîê Admin - API Key Management

GET /api/admin/api-keys üîí Admin Only

List all API keys.

POST /api/admin/api-keys üîí Admin Only

Create a new API key.

Request Body (JSON):
name string required
Descriptive name for the API key
Example Request:
curl -X POST http://localhost:5000/api/admin/api-keys \
  -H "Content-Type: application/json" \
  -b cookies.txt \
  -d '{"name": "Node-RED Production"}'
DELETE /api/admin/api-keys/{id} üîí Admin Only

Delete an API key.

PATCH /api/admin/api-keys/{id}/revoke üîí Admin Only

Revoke (disable) an API key.
